تدفق العمل الكامل
1
مدير الموقع

إعداد الموقع من داشبورد إدارة الموقع

يسجل دخول بحساب خاص بالموقع (مش نظام الإنتاج) ← يضيف صفحات (من نحن، خدمات) ← يضيف أصناف الكتالوج (صور + وصف + مواصفات — بدون سعر) ← يتحكم في كل محتوى الموقع

2
الزائر / العميل المحتمل

تصفح الموقع + طلب عرض سعر

يدخل الموقع ← يتصفح الصفحات والكتالوج ← يختار أصناف + يحدد كميات ← يملأ بياناته (اسم، شركة، تليفون، إيميل) ← يرسل طلب عرض سعر (RFQ)

3
مدير الموقع

استلام الطلب في داشبورد الموقع

يشوف طلبات عروض الأسعار الواردة ← يبلّغ فريق المبيعات في نظام الإنتاج

4
فريق المبيعات (نظام الإنتاج)

التسعير + إرسال العرض

يسحب بيانات الطلب عبر Internal API ← يسعّر الأصناف ← يطبع PDF عرض السعر ← يبعته للعميل (واتساب / إيميل)

5
تحويل تلقائي

تحويل RFQ ← أوردر في نظام الإنتاج

لو العميل وافق ← يتحول Lead لعميل رسمي ← يتحول RFQ لأوردر ← يكمل الدورة العادية (إنتاج ← استلام ← تسليم ← فاتورة)
